# 🏥 AI Enabled Conversational IVR System

## 📌 Project Overview
This project is an AI-powered IVR system designed for hospital navigation assistance. It allows users to interact using voice or text and get directions to hospital departments.

---

## 🚀 Features
- 🎤 Voice-based interaction
- 🌐 Multilingual support (English, Hindi, Telugu)
- 🧠 AI-based intent detection
- 🔊 Text-to-Speech responses
- 📞 IVR simulation UI with keypad
- 🏥 Hospital navigation assistance

---

## 🛠 Tech Stack
- Backend: FastAPI (Python)
- Frontend: HTML, CSS, JavaScript
- AI Services:
  - Speech-to-Text
  - Text-to-Speech
  - Translation API
