# 🚀 AI Enabled Conversational IVR – Hospital Navigation Assistance

## 📌 Project Overview

This project is an **AI-enabled Conversational IVR system** designed to assist users with hospital navigation. It allows users to interact using voice input and receive information such as department locations, doctor availability, and general hospital guidance.

### Backend (Local Server)

The backend of this project is built using **FastAPI** and runs locally.

⚠️ Note:

* The backend is not deployed online.
* Features like **“Start Call”** require the backend server to be running locally.

---

## ⚙️ How to Run the Project Locally

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Harshasri070/Hospital-Navigation-Assistance-IVR.git
cd your-repo
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the backend server (FastAPI)

```bash
uvicorn main:app --reload
```

### 4️⃣ Open frontend

* Open `index.html` from the `frontend` folder
  OR
* Run using a local server for better experience

## 🧠 Features

* 🎤 Voice-based interaction
* 🌐 Multilingual support (English, Hindi, Telugu)
* 🏥 Hospital department navigation
* 🤖 AI-based response handling
* 📞 IVR-style call simulation

---

## ⚠️ Important Note

* The deployed frontend works for UI demonstration.
* The **call functionality will not work on the deployed site** because:

  * It depends on a locally running backend (`http://127.0.0.1:8000`)
* To test full functionality, run the backend locally using FastAPI.
